# Libft

## Description

La **Libft** est une bibliothèque C développée dans le cadre du cursus de l'école 42.
Elle regroupe des fonctions standard de la bibliothèque C ainsi que des fonctions utilitaires, offrant une base solide pour le développement d'autres projets.
C'est une base essentielle et réutilisable pour construire des programmes dans de futurs projets.

---

## Fonctionnalités

### Partie 1 : Fonctions Libc
Les réimplémentations des fonctions standard de la libc :
- Gestion de chaînes de caractères (`ft_strlen`, `ft_strcpy`, `ft_strchr`, etc.).
- Gestion de la mémoire (`ft_memset`, `ft_memcpy`, etc.).
- Manipulation de listes simples (`ft_lstnew`, `ft_lstadd_front`, etc.).

### Partie 2 : Fonctions additionnelles
Des fonctions utilitaires conçues pour faciliter le développement :
- Conversion (`ft_itoa`, `ft_atoi`, etc.).
- Manipulation de chaînes avec allocations (`ft_substr`, `ft_strjoin`, etc.).

### Bonus : Gestion des listes chaînées
Implémentation d'opérations sur des listes chaînées pour une gestion plus flexible des structures de données.
